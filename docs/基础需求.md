#### **一、 基础功能**

**核心目标：** 实现芯片引脚的“可视化”与“手动记录”数字化，替代传统的表格记录方式。

- **芯片封装可视化**
  - 在网页上绘制芯片的封装图（如 LQFP64），直观展示物理引脚布局 。
  - 默认自动标记芯片的基础固定引脚，包括电源（VDD/VSS）、复位（NRST）、启动模式（BOOT）等 。
- **引脚功能定义（交互）**
  - 支持用户通过单击或双击引脚进行操作（防误触设计） 。
  - 点击引脚后弹出功能菜单，展示该引脚所有可复用的功能选项（如 GPIO、ADC、USART 等），供用户手动选择 。
- **状态标记与反馈**
  - 提供明确的视觉反馈，当引脚被配置后，引脚颜色发生变化或显示图钉图标进行标记 。
  - 用户可以手动勾选“标记该引脚已被使用”，便于直观区分空闲与占用资源 。

#### **二、 进阶功能**

**核心目标：** 加入“外设逻辑”与“工程管理”，解决人工查表慢、易冲突的痛点，接近专业工具体验。

- **外设导向配置 (Peripherals View)**

  - 增加外设资源列表面板，列出芯片支持的所有外设（如 UART, SPI, I2C） 。
  - **自动映射：** 用户在列表中选中某个外设时，系统自动在芯片图上高亮标记该外设对应的物理引脚 。
  - **预选预览：** 鼠标悬停在外设或功能上时，提供引脚位置的高亮预览功能 。

- **智能冲突检测**

  - 自动识别外设之间的引脚冲突。当一个外设占用了特定引脚后，其他需要复用该引脚的外设将自动变为“不可选”状态，防止资源打架 。

- **工程配置文件管理**

  - **状态保存（导出）：** 支持将当前的引脚分配方案导出为配置文件 。

  - **进度恢复（导入）：** 支持拖拽配置文件到网页，恢复之前的分配进度，方便项目升级修改或多人协作 。
